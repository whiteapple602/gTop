# gtop

<!-- ![screen record](img/demo.gif) -->

![screen record](https://raw.githubusercontent.com/aksakalli/gtop/master/img/demo.gif)

System monitoring dashboard for terminal.

**Intalling:**

```
$ npm install gtop -g
```

You can sort process table by pressing

* `p`: Process Id
* `c`: CPU usage
* `m`: Memory usage
